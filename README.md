# Desafio - Otimizando o sistema bancário com funções Python

Trilha Python LuizaLab + DIO

## Proposta

Criar funções para ações necessárias à criação de contas e registro de cliente, como também, operações monetárias do cliente junto ao banco. 
No código inicial essas ações são feitas diretamente nas condições (if/elif)
Usei tudo o que aprendi até o momento e fiz consultas com IA e na internet para entender melhor como chegar a resolução de problemas no código.
